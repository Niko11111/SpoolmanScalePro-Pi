⚠ Danger Zone

These actions are irreversible. Make sure you have a backup before proceeding.

Delete Spoolman Database
Deletes the Spoolman database file. The Docker image stays installed. Spoolman will restart with an empty database.
Delete FilaMan Database
Deletes the FilaMan database file. The Docker image stays installed. FilaMan will restart with an empty database.
Remove Spoolman completely
Stops and removes the Spoolman container and image. All data will be deleted. Must be downloaded again on next activation.
Remove FilaMan completely
Stops and removes the FilaMan container and image. All data will be deleted. Must be downloaded again on next activation.
